Privacy Policy

Your privacy is absolute. Because our software runs entirely inside your local browser tab, your hardware data never leaves your hand.

1. Scope of This Policy

This Privacy Policy governs the web application hosted at mobiledevice.tools (the "Site"). It describes how we manage hardware integrations, browser permissions, and telemetry logs while you utilize our client-side test suite.

2. Client-Side Processing Paradigm

Unlike traditional diagnostics platforms, our system operates strictly as a decoupled, client-side browser script execution model:

  • No Server Collection: Any media streams, coordinate matrices, or sensor frequencies captured during diagnostic cycles reside transiently inside your device's runtime environment memory allocation space.
  • Zero Storage Overhead: We do not deploy databases, backend application servers, or persistent log collectors designed to parse device identifiers, serial numbers, or behavioral signatures.

3. Hardware Peripheral Access Definitions

To run specific diagnostic checks successfully, the application requests standard Web APIs through browser-level consent alerts. These include:

  • Camera & Flash Module: Evaluates image capture, color balance, and flash synchronization. The local viewports remain encapsulated inside the DOM interface element and self-terminate upon tab closure.
  • Microphone Input: Maps frequency thresholds to monitor hardware fidelity. Raw audio stream packets are discarded instantly following local calculation metrics.
  • Sensor Registries (Accelerometer, Gyroscope): Accesses device orientation telemetry variables directly via standard JavaScript event hooks.
  • Geolocation Sandbox: Queries latitude/longitude coordinates through browser parameters strictly to test GPS module responsiveness.

4. Analytics and Telemetry

To optimize platform reliability across distinct mobile rendering engines, we may record strictly anonymous platform properties (e.g., User-Agent strings, viewport specifications, or error exceptions). These strings lack personally identifiable traits and help us scale structural layouts gracefully.

5. Revisions and Adjustments

We reserve the right to tweak this layout matrix as underlying Web APIs adjust across industry specifications. Continued usage of our framework implies full acknowledgment of these operational paradigms.
