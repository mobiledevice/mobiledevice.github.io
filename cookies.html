Storage Matrix

Cookies Policy

Transparency regarding how local device states and text values are handled inside your web browser sandbox.

1. Does This App Drop Tracking Cookies?

No. We purposefully do not implement third-party ad retargeting platforms or marketing user tracking mechanisms that record behaviors across other browser properties.

2. Utilized Local Storage Mechanisms

Instead of server-side data tracking pipelines, we might leverage client-side browser storage options (such as localStorage or sessionStorage keys) solely to track structural preferences during active troubleshooting sessions:

Storage Key Type Purpose
active-filter Session State Retains selected filter pill UI toggles across components.
theme-preference Local Storage Preserves dark/light editor preference arrays.

3. Browser Override Capabilities

You can seamlessly wipe, audit, or block storage keys at any time via your browser settings panel. Disabling local storage mechanics won't break our hardware script bindings—it just resets your visual system configurations back to default dark/light states upon refresh.
